Cornerstone Widget Examples DICOM image

This is an example of how to use the Cornerstone widget to load a DICOM image using the CornerstoneWADOImageLoader

Go back to the Examples page

Controls

  • Left click and drag - Adjust the window width and window center of the image
  • Pan by dragging the middle mouse button
  • Zoom by dragging the right mouse button
  • Zoom by rolling the mouse wheel
