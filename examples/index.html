Cornerstone Widget Examples

These examples are meant to demonstrate the usage of the Cornerstone Widget. They are written such that they can be opened directly in your web browser from github or your local repository. Use the "view source" capabilities built into your browser to see how each example works.

  • dicomImage - The widget loads a DICOM image using the CornerstoneWADOImageLoader
  • webImage - The widget loads a DICOM image using the CornerstoneWebImageLoader
